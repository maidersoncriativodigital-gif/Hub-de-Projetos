# O Meu Lab de Código 🚀

Bem-vindo ao **Meu Lab de Código**, o repositório central dos meus estudos e portfólio de desenvolvimento web. Este projeto documenta o meu progresso desde os primeiros passos até à criação de projetos estruturados, focando-me em Front-End.

## 🎯 Sobre o Projeto

Esta página principal funciona como um *hub* que reúne todos os meus mini-jogos, páginas temáticas e experiências de código. Foi construída com uma interface em "dark mode", design totalmente responsivo e animações de scroll suaves.

## 📂 Projetos Incluídos

Neste repositório, vai encontrar os seguintes projetos de estudo, organizados em pastas individuais:

* **Mundo-Mini:** Landing page para um estúdio artístico (Design responsivo e animações).
* **Harry-Potter:** Galeria e landing page com ilustrações do universo de Hogwarts.
* **Carnaval-de-Mascaras:** Página interativa com verificação de entrada temática e modals.
* **Caverna-das-Tres-Escolhas:** Mini-aventura de texto/narrativa com suporte a teclado.
* **Portal-Magico:** Portal temático interativo baseado em inputs de palavras mágicas.
* **Receitas:** Site de receitas prático com layout otimizado para mobile (Mobile First).
* **Jogo-da-Velha:** O clássico Jogo da Velha com lógica de jogador vs jogador .
* **Justify:** Sistema de gerenciamento de faltas, apresentado em um trabalho de TCC.

## 📋 Instruções Específicas dos Projetos

Alguns projetos deste repositório possuem configurações específicas (como dados de login, dados de teste, seeders, etc.). 

Pode consultar o **[Índice de Instruções](PROJECT-INSTRUCTIONS/INDEX.md)** para aceder rapidamente à documentação de:
- Justify
- Carnaval de Máscaras
- Jogo da Velha

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização, Flexbox, CSS Grid, variáveis CSS e animações.
* **JavaScript (Vanilla):** Lógica de jogos, manipulação de DOM e `IntersectionObserver`.

## 💻 Como Executar o Projeto

1. Faça o clone deste repositório ou faça o download em formato ZIP.
2. Extraia os ficheiros e abra a pasta principal.
3. Dê um duplo clique no ficheiro `index.html` na raiz do projeto para abri-lo no navegador.
4. Clique nos cartões para navegar para cada mini-projeto.
