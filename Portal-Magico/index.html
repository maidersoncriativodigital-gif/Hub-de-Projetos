O Portal Mágico

Você está diante de um portal mágico que leva a quatro reinos diferentes.
Para atravessá-lo, você deve digitar uma palavra secreta.

Como funciona
  • Clique em Entrar no Portal para abrir o formulário.
  • Digite a palavra mágica correspondente a um dos reinos: fogo, agua, vento ou terra.
  • A palavra é normalizada automaticamente (acentos e maiúsculas são ignorados).
  • Se a palavra estiver correta, a página altera a imagem, o fundo e exibe uma mensagem do reino.
  • Para fechar o formulário, use o botão Cancelar, o botão ×, pressione Esc ou clique fora do modal.
